# SafeHome IoT Mobile v1.0.1

Primeira versao publica do aplicativo mobile do ecossistema SafeHome IoT.

## Principais recursos

- Autenticacao e cadastro de usuarios.
- Configuracao inicial da residencia.
- Dashboard para acompanhamento dos sensores.
- Visualizacao de alertas, atividades e relatorios.
- Cadastro e gerenciamento de gateways e dispositivos.
- Configuracoes da conta e da residencia.
- Integracao com a API SafeHome IoT.
- Suporte a notificacoes locais e modo de demonstracao.

## Compatibilidade

- Expo SDK 54.
- Android, iOS e web por meio do Expo.
- API SafeHome IoT v1.0.1.
- Firmware SafeHome IoT v1.0.1.

## Qualidade e distribuicao

- Projeto configurado com TypeScript.
- Codigo-fonte distribuido sob a licenca MIT.
- Licencas das dependencias documentadas em `THIRD_PARTY_NOTICES.md`.

## Configuracao

Copie `.env.example` para `.env` e configure `EXPO_PUBLIC_API_BASE_URL` com o
endereco da API acessivel pelo dispositivo.
